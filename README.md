# 日本の人口ピラミッド 1945-2050

1945年から2050年までの日本の人口構造をインタラクティブに可視化するWebアプリです。

## データ出典
- **実績値（1945〜2020年）**：総務省統計局 国勢調査
- **推計値（2025〜2050年）**：国立社会保障・人口問題研究所 日本の将来推計人口（2023年推計）

## 表示指標
- 5歳階級別・男女別人口ピラミッド
- 総人口（男女別）
- 平均年齢（男女別）
- 合計特殊出生率
- 高齢化率 / 生産年齢人口率 / 年少人口率

## ローカル実行

```bash
npm install
npm run dev
```

## デプロイ

`main` ブランチにプッシュすると GitHub Actions が自動でビルド・デプロイします。

事前に GitHub リポジトリの Settings → Pages → Source を **GitHub Actions** に変更してください。
